# JGGL — CSV

## Основной способ (рекомендуется)

В дашборде открой **JGGL** → блок **«Загрузить CSV»**:
1. Выбери **Авто по датам CSV** (или Waitlist / Redirect)
2. Выбери `.csv` / `.xlsx` файл
3. Готово — данные попадут в лист месяца (Июн / Июл …)

Статистика сохраняется в облако (JSONBin) и видна всем на дашборде.

Формат Meta Ads / обычный CSV:

```csv
Date,Spend,Reg,cost Reg,Impressions,Clicks,CTR,CPM,CPC
2026-07-01,100.5,12,$8.38,2000,150,7.5%,50.25,0.67
```

Waitlist / Redirect по-прежнему из Google Sheets.
