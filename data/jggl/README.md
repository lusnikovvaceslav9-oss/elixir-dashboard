# JGGL — CSV

## Основной способ (рекомендуется)

В дашборде открой **JGGL** → блок **«Загрузить CSV»**:
1. Выбери **Авто iOS / Android** (по умолчанию)
2. Выбери `.csv` / `.xlsx` файл Meta Ads
3. Готово — строки разложатся по листам **iOS** / **Android**

Правило:
- в названии кампании / ad set / ad есть `android` (без учёта регистра) → **Android**
- иначе → **iOS**

Месяц выбирается переключателем сверху (Июн / Июл …). Waitlist / Redirect / Web — как отдельные листы.

Статистика сохраняется в облако (JSONBin) и видна всем на дашборде.

Формат Meta Ads:

```csv
Date,Campaign name,Spend,Reg,cost Reg,Impressions,Clicks,CTR,CPM,CPC
2026-07-01,JGGL_iOS_...,100.5,12,$8.38,2000,150,7.5%,50.25,0.67
2026-07-01,JGGL_Android_...,80.0,8,$10.00,1500,100,6.6%,53.3,0.80
```
