# JGGL — CSV

## Основной способ (рекомендуется)

В дашборде открой **JGGL** → блок **«Загрузить CSV»**:
1. Выбери лист: `App` (или Web / своё имя)
2. Выбери `.csv` файл
3. Готово — появится в разбивке

Файл хранится в браузере (localStorage). На другом компьютере нужно загрузить снова.

Формат:

```csv
Date,Spend,Reg,cost Reg,Impressions,Clicks,CTR,CPM,CPC
2026-07-01,100.5,12,$8.38,2000,150,7.5%,50.25,0.67
```

Waitlist / Redirect по-прежнему из Google Sheets.

## Опционально (репо)

Можно также класть файлы сюда + `manifest.json` — подхватятся автоматически после деплоя.
